# VWeaponEditor
An app i'm working on, which lets you edit weapons and vehicle weapon meta data live in GTA5 as an external tool

I'm making this just for educational purposes, and also for fun. I love editing the weapon/vehicle configs to absurd levels, like
shooting 20 missiles per second from a jet. But to edit things like that, you need to restart your game (unless there's a live editor for vehicle weapons that
I haven't found... there's a few for guns but not for vehicle weapons)


I will not be making this app to support online/multiplayer, because GTA Online already has enough cheaters
